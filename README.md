# 🇮🇷 Antigravity & VS Code Persian RTL & Vazirmatn Fix

مجموعه **اسکیل (Skill)** و **اسکریپت تک‌کلیکی** برای **راست‌چین‌سازی کامل (RTL)**، **نصب خودکار فونت وزیرمتن (Vazirmatn)**، **تنظیم تم Slate Dark** و **اصلاح پاسخ‌های هوش مصنوعی (Antigravity AI Agent)** در ادیتور VS Code و Antigravity.

---

## ⚡ ویژ‌گی‌های اصلی (Features)

* 🔤 **نصب خودکار فونت وزیرمتن (Vazirmatn):** دانلود و نصب کامل تمام وزن‌های فونت به صورت سراسری در ویندوز.
* 📐 **راست‌چین‌سازی هوشمند (RTL + Bidi):** اصلاح کامل متون فارسی، علائم نگارشی و پرانتزها در ادیتور و چت.
* 💻 **کادرهای اختصاصی لاگ و کد (LTR Log Viewers):** قرارگیری خودکار لاگ‌ها، مسیر فایل‌ها (<span dir="ltr" style="display:inline-block; font-family:Consolas; background:rgba(255,255,255,0.08); padding:2px 6px; border-radius:4px;">C:\...</span>) و کدها در کادرهای چپ‌به-راست استاندارد.
* 🎨 **تم اختصاصی Slate Dark:** تنظیم تم تاریک ملایم در <span dir="ltr" style="display:inline-block; font-family:Consolas; background:rgba(255,255,255,0.08); padding:2px 6px; border-radius:4px;">settings.json</span> جهت کاهش خستگی چشم.
* 🤖 **اسکیل و قوانین سراسری (Antigravity Skill & Rules):** قابلیت فعال‌سازی مستقیم به عنوان اسکیل در برنامه Antigravity.

---

## 🚀 روش‌های نصب و استفاده (Installation)

### روش اول: اجرای تک‌خطی در پاورشل (PowerShell)
```powershell
irm https://raw.githubusercontent.com/T4wroot/antigravity-persian-fix/main/install.ps1 | iex
```

### روش دوم: فعال‌سازی مستقیم به عنوان اسکیل (Antigravity Skill)
